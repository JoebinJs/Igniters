Projects

  • Uncomplicated Clap Automation

    Avoiding Electric Shocks

    Our innovative project provides a novel solution for safeguarding children from electric shocks while simultaneously adding an element of fun and engagement. By simply clapping their hands, children can activate the system, which not only prevents potential electric shocks but also plays their favorite songs. This dual functionality not only enhances safety but also fosters enjoyment and interaction, making it an invaluable addition to any household or childcare environment. With its user-friendly design and entertaining features, our project offers a seamless blend of safety and entertainment, ensuring peace of mind for caregivers while delighting children with their favorite tunes at the clap of their hands.   • Cellular Device Detection

    Involves planning and equipment installation for monitoring. Continuous analysis identifies unauthorized devices, triggering responses like signal blocking or alerting security personnel for effective security measures. (February 2023)

  • IoT Smart Security

    GSM Module Integration

    An IoT-based Smart Security and Surveillance System utilizes a GSM module for communication and integrates sensors for monitoring. It enables remote access and alerts via mobile networks, enhancing security with real-time updates and timely threat responses. (August 2023)

  • Power Monitoring System

    IoT Solution

    Leveraging ESP32, the power monitoring system tracks distribution transformer performance with integrated sensors for fault detection and environmental monitoring. Real-time data, including GPS info, is displayed on an LCD, while cloud-hosted webpages provide remote access for comprehensive oversight. This IoT solution combines ESP32 technology with sensor integration to ensure efficient monitoring and enhanced transformer functionality.
